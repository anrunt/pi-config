# Pi config

Personal configuration for the Pi coding agent.

Tracked:
- `agent/settings.json`
- custom extensions in `agent/extensions/`
- custom skills in `agent/skills/`
- Pi package manifest/lockfile in `agent/npm/`

Ignored:
- credentials (`agent/auth.json`)
- session history (`agent/sessions/`)
- local todos/runtime state
- installed dependencies and binaries
